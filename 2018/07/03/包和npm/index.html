包和NPM

1、包结构

从包的实质和包含有的成分入手简单的介绍包的属性以及特点。

包的实质：存档文件。即一个目录直接打包为.zip或者tar.gz格式的文件，安装解压后就会还原成目录。

包的属性：package.json ：包描述文件。

bin ：用于存放可执行二进制文件的目录。

lib : 通常情况下是为了保存JS文件。

doc : 主要用来保存文档的目录。

test : 存放单元测试用例的代码。

a.包描述文件 (package.json)

主要是为了详细的介绍该包。通常情况下它的存在主要是解释说明。

2、npm常用功能

作用： npm对于Node,它可以帮助完成第三方模块的发布、安装和依赖。

帮助命令：$ npm -v //查看当前系统中npm的版本号

$ npm

Usage: npm <command>  //查看常见的命令

$npm  help <command>  //可以查看具体的命令说明

安装包依赖： $ npm install express -g //全局安装

$ npm  install  module_name  -g 

(本地安装和镜像安装)
