发布搭建属于自己的npm包

Mar 25, 2018后端点击

博文概要

介绍npm包管理工具的作用以及如何快速搭建专属自己的npm包。

什么是NPM？

NPM是随同NodeJS一起安装的javascript包管理工具，能解决NodeJS代码部署上的很多问题。

允许用户从NPM服务器下载别人编写的第三方包到本地使用。
允许用户从NPM服务器下载并安装别人编写的命令行程序到本地使用。
允许用户将自己编写的包或命令行程序上传到NPM服务器供别人使用。

前期准备

1、注册一个npm账号(NPM官网注册即可)

2、创建一个简单的包

    本地创建一个项目文件夹smalllemon_demo (避免和NPM上已有的包名重复冲突就好) 然后通过终端进入文件夹。

    代码操作命令： 
                cd smalllemon_demo  //通过终端进入文件夹

                npm init            //包信息管理文件package.json


                接着在sugars_demo文件夹里创建一个index.js文件

                  ;(function (global, factory) {
                    typeof exports === 'object' && typeof module !== 'undefined' ? module.exports = factory() :
                    typeof define === 'function' && define.amd ? define(factory) :
                        global.moduleName = factory()
                  }(this, (function () {
                    var test = {
                      sayHi: function () {
                        console.log('hi');
                      }
                    };
                    return test
                  })))


3、开始发布创建好的包

     npm adduser   //输入前面注册好的NPM账号，密码和邮箱，将提示创建成功

     npm publish   //接着先进入项目文件夹下，然后输入以下命令进行发布

     myHP-PC:smalllemon_demo smalllemon$ npm publish
     + smalllemon_demo@1.0.0    //代表版本号为1.0.0的包发布成功

     npm ERR publish 403      //代表版本号为1.0.0的包发布失败

4、更新已经发布的包

    npm publish 

   (注：每次更新时，必须修改版本号后才能更新)
