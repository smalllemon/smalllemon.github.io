Module

1、Module

概念：弥补了JS没有模块这个缺陷，它要求严格，命令主要由两个关键字构成：

      export:用于规定模块的对外接口
      import:用于输入其他模块提供的功能

用法：模块是独立文件，没有使用export关键字时就无法获取到该变量。

      用法一   export var filename = "XXXX";

      用法二   var filename = "XXX";

               var writer = "lll";

               export{ filename,writer }

      export不仅可以单独输出一个变量，而且可以输出一组变量。

      用法三   function show(){}

               export{show  as  vw}

      export输出了一个函数并使用了关键字AS进行重新命名
