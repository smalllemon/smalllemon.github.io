使用vue全家桶的工程

Mar 30, 2018前端框架点击

总结vue全家桶使用注意事项

1、 搭建自己的项目需要使用到的工具：vue-cli vue-resoure vue-router vuex

2 、项目工程目录使用脚手架搭建好之后，完善整个结构，需要部署的结构有：

组件存放的目录    components

页面存放的位置    pages

外部的静态资源    assets

路由存储的位置    router/index.js

集中管理全局状态  store/ actions.js / getters.js / index.js / mutations.js

3、全局注册模块的方法以及使用的方法

修改位置：main.js

相关操作：以axios为例：

　　　　　　 import axios from “axios” //全局引入axios

Vue.prototype.$http = axios;  //全局注册方法

axios.defaults.headers.post['Content-Type'] = 'application/x-www-form-urlencoded;charset=UTF-8';

4、路由的相关操作

安装模块 npm install vue-router

导入模块 import Vue from ‘vue’

import Router    from 'vue-router'


Vue.use(Router)

路由设置 具体的模块

{
    path:'/pageHome',

    name:'pageHome',

    component : resolve => require.ensure([], () => resolve(require('@/pages/pageHome.vue')), 'pageHome')

},

路由定向 设置默认加载的页面

     {
      path: '/',

      redirect :'/pageHome'

},

　　
5、在开发组件过程中遇到的问题总结

提取组件需要封装的数据:子组件通过prop向父组件发出数据请求

父组件调用子组件:(1)、在JS标签内需要引入该组件

　　
(2)、在component对象中注册该组件

(3)、在页面中以元素标签的方法引入该组件并向它传值
