Vue全家桶介绍

Vue有著名的全家桶系列，包含了vue-router(http://router.vuejs.org),vue-resource(https://github.com/pagekit/vue-resource),vuex(http://vuex.vuejs.org), 构建工具vue-cli以及sass样式,就是一个完整的vue项目的核心构成。

概括起来就是：
1.项目构建工具
2.路由
3.状态管理
4.http请求工具

1、vue-cli 项目脚手架

作用： 构建前端项目工程,降低了webpack的开发难度,支持热更新,有webpack-dev-server的支持，启动了一个请求服务器，搭建了一个测试环境，只关注开发就好。

流程：

npm  install  vue-cli -g    //全局安装vue-cli 

vue   init  webpack  project_name  //初始化，生成项目目录

Cd   project_name       //进入根目录文件

Npm   install            //安装包依赖  

npm  run dev             //运行项目

2、vue-resource

作用：主要是为了加载资源。

方式：this.$http().then(res){}

解析：包含请求参数和响应参数，并且在then 部分中对响应资源（response）进行处理或者格式化

类比：它在全家桶中的作用类似于JS中的ajax，作为前端和后台数据交互的桥梁，促进前后两端的数据 交互。

使用axios中间件：npm install axios –save

(注：使用前必须在main.js中全局注册： Vue.prototype.$http = axios;)

实例代码：axios.get(API).then(function(response){
console.log(response);
})
.catch(function(err){
console.log(err);
});

3、vue-router

作用：通过切换组件内容进行页面的刷新。

安装：npm install vue-router

格式：{path:’/home’, component: ‘home’}

使用方法：

import Vue  from'vue' 

import  VueRouter  from'vue-router'

Vue.use(VueRouter)

解析：创建组件模板，引入vue-router.js ——>创建一个router对象——>映射路由——>使用
v-link命令（触发点击跳转的命令）——>使用标签（填充视图的命令）——>启动命令（router.start()）

4、vuex

作用：全局的数据管理

构成部分：state、action、mutation、getters组成
