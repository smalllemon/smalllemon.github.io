浅谈Promise对象

1、Promise产生的原因

示例代码：setTimeout(function(){
left(function(){
setTimeout(function(){
left(function(){
setTimeout(function(){
left();
},2000);
});
}, 2000);
});
}, 2000);

分析：如上述代码所示，多层业务进行了嵌套，不仅阅读困难，而且不利维护。

此时，Promise应运而生。

2、Promise介绍

概念：它是一个对象，用来传递异步操作的消息。它代表了某个未来才会结果的事件。

特点：
(1)、状态不受外界的影响。

     Pending（进行中）

     Resolved（已完成，又称 Fulfilled）

     Rejected（已失败）

     (注：只有异步调用返回的结果才能使它的状态发生改变。)

(2)、一旦状态改变，就不会再变。

          Pending——>Resolved

          Pending——>Rejected

    (注：无论发生上述那一种改变，状态就会凝固，不会再变)

3、Promise基本语法

var promise = new Promise(function(resolve, reject) {
    // ... some code
    if (/* 异步操作成功 */){
      resolve(value);
    } else {
      reject(error);
    }
});

实例一:

function loadImageAsync(url){
    return newPromise(function(resolve,reject){
        var image = new Image();
        image.onload  = function(){
          resolve(image);
        };

        image.onerror = function(){
          reject(new Error("Could not load image"));
        };

        image.url = url;
    });
}

实例二：

var getJson = function(url){
    var promise = new Promise(function(resolve,reject){
        var client = new XMLHttpRequest();
        client.open("GET",url);
        client.onreadystatechange = handler;
        client.send();

        function handler(){
          if(this.readyState !== 4){
              return;
          }
          if(this.status ==200){
              resolve(this.response);
          }else{
              reject(new Error(this.statusText));
          }
        };
    })

    return  promise;
};

getJson(url).then(function(json){
    console.log("Contents:" +json);
},function(error){
    console.error("出错了", error);
});

4、Promise.prototype.then()

作用:为Promise实例添加状态改变时的回调函数。

使用该方法可以指定一组按照次序调用的回调函数。

getJson(api).then(function(argument)){
return getJson(argument.URL);
}).then(function success(res){
console.log(“resolved: “, res);
},function error(res){
console.log(“rejected: “,res);
});

5、Promise.prototype.catch()

作用:处理产生错误时的回调函数。

代码实例：
getJson(api).then(function(argument)){
return getJson(argument.URL);
}).catch(function(error){
console.log(“wrong “, error);
});

6、Promise.all() (&&)

作用:将多个Promise实例，包装成一个新的Promise实例。

实例：var p = Promise.all([p1, p2, p3]);

p要变成Resolved状态需要p1，p2，p3状态都是Resolved
如果p1,p2,p3至少有一个状态是Rejected，p的状态就变成Rejected。

作用:返回一个新的、状态为rejected的Promise实例。
